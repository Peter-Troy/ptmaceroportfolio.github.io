PDF to Word Converter App

Introduction

In today's digital world, converting PDF documents to editable Word files remains a common challenge. Meet our solution - the PDF to Word Converter, a powerful Python application that bridges the gap between scanned documents and editable text.

Chapter 1: The Problem

Every day, professionals struggle with two types of PDFs: digital PDFs with selectable text, and scanned PDFs that are essentially just images of text. This application solves both problems with a simple, user-friendly interface.

Chapter 2: The Technology Stack

Built with Python's Tkinter for the interface, this application combines several powerful libraries:

  • PyPDF2 – for digital PDF text extraction

  • pytesseract and pdf2image – OCR (Optical Character Recognition)

  • python-docx – Word document creation

  • Poppler – the PDF rendering engine

A diagram showing how the components interact in the PDF-to-Word text extraction and conversion process:

Chapter 3: Key Features

Dual Processing Modes

  • Text Extraction for digital PDFs (using PyPDF2)

  • OCR Mode for scanned documents (using Tesseract)

Intuitive Interface

  • File browsing with native dialogs

  • Progress tracking with real-time updates

  • Clear status notifications

Smart Output Handling

  • Automatic output filename suggestion

  • Proper page separation in the Word document

Chapter 4: How It Works

1. User selects a PDF file and output location

2. System determines processing mode (text or OCR)

3. For digital PDFs, it Extracts text directly using PyPDF2

4. For scanned PDFs, it Converts pages to images and Processes each image with Tesseract OCR

5. Structures content with page markers

6. Saves to a well-formatted Word document

Chapter 6: Real-World Applications

This tool is useful for:

  • Lawyers converting legal document

  • Students digitizing textbook excerpts

  • Archivists preserving historical documents

  • Businesses processing scanned form

Chapter 7: Future Enhancements

  • Batch processing multiple files

  • Advanced formatting options

  • Cloud storage integration

  • Handwriting recognition

import tkinter as tk
from tkinter import filedialog, messagebox, ttk
import PyPDF2  # For extracting text from digital PDFs
from pdf2image import convert_from_path  # For converting PDF pages to images (for OCR)
import pytesseract  # For OCR (extracting text from images)
from docx import Document  # For creating Word documents
import os  # For file path operations


class PDFExtractorApp:
    def __init__(self, root):
        # Initialize the main app window
        self.root = root
        self.root.title("PDF to Word Converter")
        self.root.geometry("600x400")

        # Configure Poppler path (for rendering PDF as images)
        self.poppler_path = r'C:/Users/Yaiba Kusanagi/Downloads/Release-24.08.0-0/poppler-24.08.0/Library/bin'

        # Verify Poppler installation
        self.poppler_available = self.check_poppler()
        if not self.poppler_available:
            messagebox.showwarning("Warning",
                                    "Poppler not found at specified path.\n"
                                    "OCR functionality will be disabled.\n"
                                    "Please download from:\n"
                                    "https://github.com/oschwartz10612/poppler-windows/releases")

        # Configure UI styles
        self.style = ttk.Style()
        self.style.configure('TFrame', background='#f0f0f0')
        self.style.configure('TLabel', background='#f0f0f0', font=('Arial', 10))
        self.style.configure('TButton', font=('Arial', 10))
        self.style.configure('TRadiobutton', background='#f0f0f0')

        # Main frame layout
        self.main_frame = ttk.Frame(root)
        self.main_frame.pack(padx=20, pady=20, fill=tk.BOTH, expand=True)

        # PDF Selection
        ttk.Label(self.main_frame, text="Select PDF File:").grid(row=0, column=0, sticky=tk.W, pady=(0, 5))
        self.pdf_path = tk.StringVar()
        ttk.Entry(self.main_frame, textvariable=self.pdf_path, width=50).grid(row=1, column=0, padx=(0, 10))
        ttk.Button(self.main_frame, text="Browse", command=self.browse_pdf).grid(row=1, column=1)

        # Processing Mode Options
        ttk.Label(self.main_frame, text="Processing Mode:").grid(row=2, column=0, sticky=tk.W, pady=(10, 5))
        self.mode = tk.StringVar(value="text")  # Default to text extraction
        ttk.Radiobutton(self.main_frame, text="Text Extraction (for digital PDFs)", variable=self.mode,
                        value="text").grid(row=3, column=0, sticky=tk.W)

        # OCR option for scanned PDFs
        ocr_radio = ttk.Radiobutton(self.main_frame, text="OCR (for scanned PDFs)", variable=self.mode, value="ocr")
        ocr_radio.grid(row=4, column=0, sticky=tk.W)

        # Disable OCR if Poppler is unavailable
        if not self.poppler_available:
            ocr_radio.state(['disabled'])

        # Output file selection
        ttk.Label(self.main_frame, text="Output Word File:").grid(row=5, column=0, sticky=tk.W, pady=(10, 5))
        self.output_path = tk.StringVar()

        # Progress bar
        self.progress = ttk.Progressbar(self.main_frame, orient=tk.HORIZONTAL, length=400, mode='determinate')
        self.progress.grid(row=7, column=0, columnspan=2, pady=(20, 10))

        # Convert button
        ttk.Button(self.main_frame, text="Convert to Word", command=self.process_pdf).grid(row=8, column=0,
                                                                                            columnspan=2, pady=(10, 0))

        # Status label
        self.status = ttk.Label(self.main_frame, text="", foreground="green")
        self.status.grid(row=9, column=0, columnspan=2, pady=(10, 0))

    def check_poppler(self):
        """Check if Poppler binaries exist at the specified path"""
        try:
            required_files = ['pdftoppm.exe', 'pdfinfo.exe']
            for file in required_files:
                if not os.path.exists(os.path.join(self.poppler_path, file)):
                    return False
            return True
        except Exception:
            return False

    def browse_pdf(self):
        """Select a PDF file and set the output path"""
        filepath = filedialog.askopenfilename(
            title="Select PDF File",
            filetypes=[("PDF Files", "*.pdf"), ("All Files", "*.*")]
        )
        if filepath:
            self.pdf_path.set(filepath)
            # Suggest output Word file path
            base = os.path.splitext(filepath)[0]
            self.output_path.set(f"{base}_extracted.docx")

    def browse_output(self):
        """Choose the output Word file location"""
        filepath = filedialog.asksaveasfilename(
            title="Save Word File As",
            defaultextension=".docx",
            filetypes=[("Word Documents", "*.docx"), ("All Files", "*.*")]
        )
        if filepath:
            self.output_path.set(filepath)

    def process_pdf(self):
        """Main PDF processing function"""
        pdf_path = self.pdf_path.get()
        output_path = self.output_path.get()

        # Validate input and output paths
        if not pdf_path:
            messagebox.showerror("Error", "Please select a PDF file")
            return

        if not output_path:
            messagebox.showerror("Error", "Please specify an output file")
            return

        try:
            # Update progress bar
            self.progress['value'] = 0
            self.status.config(text="Processing...", foreground="blue")
            self.root.update_idletasks()

            # Choose processing mode (text or OCR)
            if self.mode.get() == "text":
                text = self.extract_text(pdf_path)
            else:
                if not self.poppler_available:
                    messagebox.showerror("Error", "OCR disabled - Poppler not found")
                    return
                text = self.extract_text_with_ocr(pdf_path)

            # Save extracted text to Word
            self.save_to_word(text, output_path)

            # Update progress and show success message
            self.progress['value'] = 100
            self.status.config(text=f"Success! Saved to {output_path}", foreground="green")
            messagebox.showinfo("Success", f"PDF successfully converted to Word document:\n{output_path}")

        except Exception as e:
            self.status.config(text=f"Error: {str(e)}", foreground="red")
            messagebox.showerror("Error", f"Failed to process PDF:\n{str(e)}")

    def extract_text(self, pdf_path):
        """Extract text from digital PDF"""
        with open(pdf_path, 'rb') as file:
            reader = PyPDF2.PdfReader(file)
            text = ""
            total_pages = len(reader.pages)

            for i, page in enumerate(reader.pages):
                text += f"--- Page {i + 1} ---\n{page.extract_text() or ''}\n\n"
                self.progress['value'] = (i + 1) / total_pages * 100
                self.root.update_idletasks()

            return text

    def extract_text_with_ocr(self, pdf_path):
        """Extract text from scanned PDF using OCR"""
        pytesseract.pytesseract.tesseract_cmd = r"C:/Users/Yaiba Kusanagi/Downloads/tesseract.exe"

        try:
            images = convert_from_path(pdf_path, dpi=300, poppler_path=self.poppler_path)
            text = ""
            total_pages = len(images)

            for i, img in enumerate(images):
                text += f"--- Page {i + 1} ---\n{pytesseract.image_to_string(img, lang='eng')}\n\n"
                self.progress['value'] = (i + 1) / total_pages * 100
                self.root.update_idletasks()

            return text
        except Exception as e:
            raise Exception(f"OCR failed: {str(e)}\nPoppler path used: {self.poppler_path}")

    def save_to_word(self, text, output_path):
        """Save extracted text to Word document"""
        doc = Document()
        paragraphs = text.split('\n\n')

        for para in paragraphs:
            if para.strip():
                if para.startswith("--- Page"):
                    doc.add_heading(para, level=3)
                else:
                    doc.add_paragraph(para)

        doc.save(output_path)


if __name__ == "__main__":
    root = tk.Tk()
    app = PDFExtractorApp(root)
    root.mainloop()
